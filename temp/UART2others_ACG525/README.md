# 高云选题3第一、二题 协议转化+测量数字信号模块
**模块简介**

该模块旨在通过cdc串口接收数据并将其转化为其他协议与测量数字信号的频率、高低电平时间与占空比，主要方法为等精度测量法。
## 目前已完成的项目
- **uart转uart、IIC、SPI、PWM、CAN**
- **测量PWM波形的频率、高低电平时间与占空比**
## 待完成项目
- **完成基于cdc的串口收发数据程序**
- **在上层模块测量周期信号的数据**
## 存在的问题
- **CAN通信可配置项目少，后续可尝试添加不定字节发的程序**
- **频率等数据有时会突然变0，可能是由于测量完成状态设置的问题（问题已得到改善）**
- **占用逻辑单元较多，可以酌情考虑降低精度，降低资源消耗（目前共占用一万个逻辑单元）**
- **低频测量不准确，考虑当频率较低时用测周法，测试发现不准确的原因是锁相环输出不稳定，当改为系统时钟时则测量正确**
- **可以尝试增加除法模块，减少资源消耗的同时保证锁相环时序**
