# 高云选题三完整模块
**模块简介**

本模块为基于国产高云FPGA的多功能信号调试仪，过上位机发送命令，实现多种协议转换与解析，并采用等精度测量法实现数字信号测量，模拟部分通过串口及以太网与上位机通信，通过DDS方式输出高精度的固定波形，手绘波形，序列波形，通过ddr3缓存串口发送的buffer模式和以太网直发的流模式实现高频率范围的adc波形采集

## 已完成的项目
- 题一：基于CDC串口(CY7C60813芯片)的协议转化，包括UART、IIC、SPI、多路PWM、CAN(**扩展7**),均可实现协议发送与协议的解析(数字信号分析、即**扩展3**)，除CAN通信的发送外均实现了有最大字节限制（可调）的可变字节收发
- 题二：基于等精度测量法与智能分组测量的数字信号测量，频率与占空比分开设计，保证计算的独立性，优化闸门的设置，兼顾精度与速度，频率测量时MHZ级的精度高于千分之5，其余情况基本无误；50HZ测量速度可控制在1s以内，占空比测量基本无误
- 题三：驱动高速ADC实现模拟信号采集、存储与传输，利用以太网（**扩展1**）实现流模式（**扩展7**）与上位机进行传输，也可以选择使用基于DDR3的串口传输，实现buffer模式（扩展7），并能够通过自主设计的上位机对收集到的数据进行频域和时域分析（**扩展4**）
- 题四：驱动高速DAC输出模拟信号，预设五种波形（正弦波、方波、三角波、锯齿波、直流），且可以通过上位机实现手绘波形的输出（扩展5）与输出序列波形（扩展2）

## 项目的特点
- 协议转化可配置选项丰富，大多数为可变字节收发，使用灵活，纯verilog编写，程序移植便捷
- 数字信号测量采用先用极低闸门粗测后分组细测的方式，兼顾了精度与速度，利用除法器进行除法操作，降低了资源消耗，保证时序正常
- 以太网流模式传输高频模拟信号，基于DDR3的buffer模式传输低频模拟信号
- 支持手绘模拟波形输出，灵活性强
- 上位机集成大部分功能，GUI界面操作人性化，帮助FPGA完成各项工作
- 可同时使用多项功能（**扩展6**），满足各种场合的需求
- 资源消耗少，占用逻辑单元不到所用开发板（ACG525）的一半
